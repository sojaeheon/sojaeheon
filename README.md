### 👋 About Me

데이터 파이프라인 구축을 중심으로, 데이터가 수집·처리·저장되어  
서비스와 분석에 활용되는 과정을 프로젝트를 통해 경험했습니다.

Kafka, Airflow, Spark를 활용한 실시간·배치 데이터 처리와  
Docker, AWS 기반 실행·배포 환경을 다뤄왔습니다.

현재는 데이터 정합성, 재처리, 모니터링 등  
안정적인 데이터 파이프라인 운영에 필요한 역량을 학습하고 있습니다.

백엔드와 인프라 경험을 바탕으로  
서비스와 데이터 시스템의 연결을 이해하는 데이터 엔지니어를 지향합니다.
