## 🚀 About Me

* 데이터가 사용자에게 실제 가치로 전달되는 서비스 흐름에 관심이 있습니다.

* Kafka, Flink, Airflow, Elasticsearch 기반 데이터 처리·검색 흐름을 경험했습니다.
* Docker, AWS EC2/ECR, Nginx, Jenkins, Kubernetes 기반 실행 환경과 배포 흐름을 경험했습니다.
* 데이터 분석과 AI 기능을 서비스에 연결하는 프로젝트를 진행해왔습니다.
